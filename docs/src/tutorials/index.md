# Tutorials

Tutorials are **learning-oriented**. They guide you through a series of steps so that you can get AGNI running and see what it can do. Start here if you are new to AGNI and want to use the code. They also illustrate the kinds of results AGNI can produce.

You can also find Jupyter notebooks which reproduce these results in the [tutorials directory](https://github.com/nichollsh/AGNI/tree/main/tutorials) and the [miscellaneous directory](https://github.com/nichollsh/AGNI/tree/main/misc) of the GitHub repo.
