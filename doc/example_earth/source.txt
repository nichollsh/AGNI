The file equ.csv contains temperature, pressure, height, and VMR values through
a column of the Earth's atmosphere. 

The data were generated using the Reference Forward Model (Dudhia+2017).
Specifically, from the equ_200km.atm file on this website: 
    http://eodg.atm.ox.ac.uk/RFM/atm/

They were converted to the propriate format for AGNI using a Python script.
